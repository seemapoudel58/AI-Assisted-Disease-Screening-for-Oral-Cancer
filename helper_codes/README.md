# Helper Codes

This folder contains utility scripts used for dataset preparation and preprocessing.

## Files
- **json2txt.py**  
  Converts `annotation.json` files into YOLO-style `.txt` annotation files.

- **binary_csv_converter.py**  
  Converts datasets into a binary CSV format suitable for MobileNetV3 classification.

## Notes
- Run these scripts before training models in the notebook.  
- Input/output paths may need to be updated based on your local setup.
