# Notebook

This folder contains Jupyter/Colab notebooks related to training and experimentation.

## Files
- **OralCancer_Pipeline.ipynb**  
  Contains end-to-end training pipelines for:
  - YOLOv8n (object detection for oral cancer screening)  
  - MobileNetV3 (binary classification)

## Notes
- Designed to run on Google Colab.  
- Requires dataset preprocessing using scripts in `helper_codes/`.
